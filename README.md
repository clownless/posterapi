# postertelegrambot

## Запуск
1. Установите необходимые библиотеки
`pip install -r requirements.txt`
2. Укажите токен телеграм-бота в файле main.py

_Пример токена:_
```
TOKEN = "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
```
3. Запустите бота
`python3 main.py`
